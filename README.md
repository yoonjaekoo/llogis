## 🧠 Rating System

Math Solved는 단순 정답 개수 대신  
유저 실력과 문제 난이도를 함께 계산합니다.

예정 알고리즘:
- Glicko-2
- Elo 기반 실험 시스템

예시:
- 어려운 문제를 맞히면 레이팅 상승 ↑
- 쉬운 문제를 틀리면 레이팅 하락 ↓
- 문제 난이도도 사용자 결과에 따라 변함
